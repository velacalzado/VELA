# VELA | Tienda de Calzado

Bienvenido al repositorio de **VELA**, tu tienda de calzado en línea.

## 📦 ¿Qué contiene este proyecto?

- El código fuente del sitio web de VELA.
- Imágenes de productos y logotipos en la carpeta `img/`.
- Archivos HTML y CSS para el diseño minimalista pastel.

## 🌐 Ver el sitio en línea

Puedes ver el sitio publicado en GitHub Pages:  
[https://velacalzado.github.io/tu-repo/](https://velacalzado.github.io/tu-repo/)  
*(Cambia `tu-repo` por el nombre real de tu repositorio)*

## 🖼️ Imágenes

Las imágenes usadas en la web están en la carpeta `img/`.

## ✍️ ¿Cómo modificar el sitio?

1. Edita los archivos HTML/CSS en la raíz del repositorio.
2. Sube nuevas imágenes a la carpeta `img/`.
3. Haz commit y push o usa el editor web de GitHub.
4. Los cambios se reflejarán automáticamente en GitHub Pages en unos minutos.
